## ⚠️ إذا استمرت المشكلة

إذا استمرت مشكلة 500 في CSS:

1. **أوقف dev server** (Ctrl+C)
2. **نظف الكاش بالكامل**:
   ```powershell
   Remove-Item -Recurse -Force .next
   Remove-Item -Recurse -Force node_modules\.cache -ErrorAction SilentlyContinue
   ```
3. **أعد تثبيت المكتبات**:
   ```powershell
   npm install
   ```
4. **شغل المشروع**:
   ```powershell
   npm run dev
   ```
